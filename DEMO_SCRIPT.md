# 2-minute demo script

## 0:00–0:15 — Problem and promise

“People do not want another skin score. They want to understand what is visible today, know the safest next step, and see whether their routine appears to be making a difference. API Skin AI turns YouCam Skin Analysis into that complete decision journey.”

Show the landing page, then point to “Consent first,” “No app-side photo storage,” and “Not medical advice.”

## 0:15–0:35 — Consent and privacy

Open the assessment. Explain that the photo selector is disabled until the user explicitly consents. Highlight:

- the app does not add its own photo storage;
- YouCam’s documented 30-day provider retention is stated before consent; and
- saved history is score-only and local to the browser.

Check consent and continue.

## 0:35–0:55 — Photo quality and safety

Choose a pre-approved clear face image. Show the resolution, format, framing, and lighting feedback. Mention the photo recipe for consistent future comparisons.

Briefly show the symptom checkbox: “If a user reports pain, bleeding, an open area, or rapid change, the product changes the plan and recommends professional review. This is an educational cosmetic assessment, not diagnosis.”

## 0:55–1:20 — Technical integration

Click “Analyse my photo.” While loading, explain:

“The browser sends the photo to our protected Next.js route. The server uses YouCam’s full asynchronous workflow: it creates file metadata, uploads to the presigned URL, creates a Skin Analysis task, polls for completion, and returns only normalized visible scores. The API key and provider URLs never reach the browser.”

If using mock mode, point to the visible “Demo data” label. For final judging, record live mode if available.

## 1:20–1:45 — Explanation and action

Scroll through indicator cards. Explain that scores use inclusive plain language and are not appearance grades. Show the targeted AM/PM routine, product-category suggestions, patch-testing guidance, and medical disclaimer.

“The API provides the vision signal. API Skin AI adds the consumer decision layer: meaning, proportionate action, and uncertainty.”

## 1:45–2:00 — Progress and close

Save the score-only check-in, open History, and show the comparison and delete-all control.

“The result is not a one-off wrapper. It is a privacy-conscious loop: understand, act, compare, and know when AI is not the right next step.”

End on the landing page with the tagline: “API Skin AI—understand today, track what changes.”

## Recording safeguards

- Use a face image with explicit permission.
- Hide bookmarks, notifications, `.env` files, terminal history, and API console credentials.
- Keep the final video under three minutes.
- Do not use unlicensed music or third-party trademarks/assets.
